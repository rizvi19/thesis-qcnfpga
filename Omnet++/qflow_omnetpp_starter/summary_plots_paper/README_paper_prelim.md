# QFlow OMNeT++ Paper-Preliminary Summary

## What these results are for
- Dynamic-load network-level evidence for the paper.
- These runs complement the hardware timing/evaluation package; they do not replace it.

## Main outputs
- `omnet_paper_summary.csv`: aggregated table by topology, algorithm, and rate.
- `blocking_vs_load_<topology>.png`: blocking probability vs request rate.
- `fidelity_cdf_highload_<topology>.png`: fidelity CDF at highest tested load.
- `utilization_heatmap_<topology>.png`: utilization snapshot heatmap.
- `scalability_snapshot.png`: node-count scalability snapshot.
