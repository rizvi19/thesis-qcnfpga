# QFlow OMNeT++ Refined Dynamic-Load Package

## Purpose
- This package is the refined dynamic-load evidence layer for the QFlow paper.
- It is intended to complement the existing hardware/timing section.
- Repetitions are now distinguished through explicit `rngSeed` overrides.
