# QFlow Scope Freeze — Phase 0

## Mandatory core path
This is the path that must be completed for a strong and defensible thesis:

- Mathematical contract frozen
- Python golden model audited
- FDPE RTL
- SKAG RTL
- PMO-GA RTL
- Unit testbenches for each mandatory module
- Top-level integration
- Synthesis report set
- Evaluation against 3 mandatory baselines
- Core figures/tables for thesis and paper

## Recommended extension path
Only start these after the mandatory core is safe:

- QP-TSN scheduler
- board deployment on Artix-7
- irregular backbone topology
- extra ablations and polished demo work

## Stretch path
Do only if time remains after the extension path:

- DRL / DQN baseline
- 32-node scale case
- expanded hardware board measurements
- journal-extension-only figures

## Hard scope rule
If a core block is unfinished, do not spend thesis-critical time on optional blocks.
