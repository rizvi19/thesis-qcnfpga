## Mathematical meaning

The LUT stores Q0.16 fixed-point values for exp(-x), where:

- x range: [0, 8)
- entries: 256
- step: 8/256 = 1/32
- value: round(exp(-x) × 65535)

The FDPE kernel computes:

fidelity_out = f_init × LUT[index]

with Q0.16 × Q0.16 multiplication and Q0.16 output truncation.

## LUT quantization summary

From `exp_lut_256_summary.txt`:

- maximum absolute quantization error: 7.602303880458e-06
- maximum relative quantization error: 1.816016214100e-02

The relative error is larger near the low-value tail of exp(-x), where the denominator is very small. For paper-quality reporting, operational-range error summaries should also be reported.
